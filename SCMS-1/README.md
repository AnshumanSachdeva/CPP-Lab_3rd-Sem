# Smart Campus Management System (SCMS)

## Overview
SCMS is a comprehensive campus management system built using Object-Oriented Programming principles in C++. It provides modules for user management, course management, authentication, and logging.

## Features
- **User Authentication**: Registration, login, password management
- **Role-based Access**: Student, Teacher, and Admin roles
- **Course Management**: Add, update, delete courses
- **Enrollment System**: Students can enroll/drop courses
- **Logging System**: Comprehensive logging of all actions
- **File Persistence**: Data saved to files for persistence
